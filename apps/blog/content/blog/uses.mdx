# My Setup

I try to maintain a list of all the stuff I use, so that it is easy to direct someone when they ask me about it here.
This list contains my software and operating system configurations for now. I might later include my hardware specs

## Programming Fonts I use

I prefer [Dank Mono](https://philpl.gumroad.com/l/dank-mono/), but I have also liked using the fonts below

- [Fira Code](https://github.com/tonsky/FiraCode)
- [Iosevka](https://typeof.net/Iosevka/)
- [Cascadia Code](https://github.com/microsoft/cascadia-code)

### Apps I use

- [Orbstack](https://orbstack.dev/) - Docker desktop alternative for mac
- [Hex](https://github.com/kitlangton/Hex) - Voice to text app
- [Spacedrive](https://github.com/spacedriveapp/spacedrive) - File explorer for mac
- [Hoppscotch](https://hoppscotch.io/)
- [VSCode](https://code.visualstudio.com/download) - Code editor

### Terminal

- [Ghostty](https://ghostty.org/) - Fash cross-platform terminal emulator
- [zsh](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH) & [Oh-My-Zsh](https://github.com/ohmyzsh/ohmyzsh)
- [btop](https://github.com/aristocratos/btop?tab=readme-ov-file#themes) - Interactive process viewer
- [tree](https://formulae.brew.sh/formula/tree) - Display directories as trees
- [lsd](https://github.com/lsd-rs/lsd) - Rewrite of `ls` with  lots of features

### VSCode Extensions

